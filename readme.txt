== Description ==

add Facebook like button to your Wordpress blog.
(more info at http://developers.facebook.com/docs/reference/plugins/like)

== Installation ==

- unzip archive to wp-content/plugins directory
  (or you can cd into your wordpress_root/wp-contents/plugins and do:
  git clone git://github.com/ayn/wp-facebook-like-button.git)
- activate it in Plugins page in wp-admin
